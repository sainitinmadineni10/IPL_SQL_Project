📌 Project Overview

This project analyzes IPL (Indian Premier League) match data using SQL and MySQL.
The goal of this project is to extract meaningful insights from IPL matches such as team performance, toss impact, scoring trends, captain records, venue analysis, and player achievements.

📊 SQL Analysis Performed

1. Team With Most Wins
Analyzed which IPL team has won the highest number of matches.

2. Most Successful Captains
Identified captains with the most IPL match victories.

3. Toss Decision Analysis
Analyzed how frequently teams chose batting or fielding after winning the toss.

4. Toss Winner vs Match Winner
Checked whether winning the toss increases chances of winning the match.

5. Highest Scoring Matches
Found matches with the highest combined team scores.

6. Team Average Score Analysis
Calculated average scores of teams across all venues.

7. Player of the Match Awards
Identified players with the most Player of the Match awards.

8. Venue Analysis
Found venues that hosted the highest number of IPL matches.

9. Season-wise Match Analysis
Calculated total matches played in each IPL season.

10. Chasing Performance Analysis
Analyzed teams with most successful run chases.

11. Home Ground Win Percentage
Calculated home win percentage for each team.

12. Closest Matches Analysis
Identified matches with the smallest run differences
